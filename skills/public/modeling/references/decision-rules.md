# Decision Rules — 经验规则与默认参数

> 给 LLM 在生成 Recipe 时直接可用的"if X 则 Y"规则。**不是教科书**，不是综述。
> 规则应支撑参数选择，源头是公开文献 / 社区惯例 / VASP-wiki / ASE / pymatgen 默认。
> 引用规则前先确认对应 builder/transform 已实装（见 `PLAN.md` 进度表）。

---

## 1. 真空层 (vacuum)

| 场景 | 推荐厚度 (Å) | 原因 |
|---|---|---|
| 静态 DFT slab | ≥ 15 | 抑制周期像间偶极/电荷耦合 |
| AIMD slab | ≥ 20 | 含液体层时还要再加水层厚度 |
| 含吸附分子的 slab | slab + adsorbate + 12 | 分子最高原子到镜像底层 ≥ 12 Å |
| 二维材料 (graphene, MoS₂) | ≥ 20 | 范德瓦尔斯 + 偶极 |
| 极性表面 (ZnO 0001) | ≥ 25 + 偶极校正 | LDIPOL=.TRUE., IDIPOL=3 |

**轴向约定**：ASE / pymatgen 默认在 c 轴加。多组分异质结时 Recipe 必须显式 `vacuum.axis`，不依赖默认。

---

## 2. Slab 层数与固定 (slab + selective_dynamics)

| 体系 | 层数 | 底层 fixed |
|---|---|---|
| 金属表面 (FCC/BCC) | 4–6 | 底 2 层 |
| 金属吸附计算 | 5–7 | 底 2–3 层 |
| 半导体 / 绝缘体 | 6–8 | 底 2–3 层 |
| 极性 / 重构表面 | 8–12 | 底 3–4 层 |
| 包含偶极的 slab | 偶数层 + 对称 | symmetric=True |

**收敛性准则**：把层数 +2，吸附能变化 < 0.05 eV / 表面能变化 < 5 mJ/m² 即可。

---

## 3. 超胞 (supercell)

| 目标 | 推荐 in-plane 矩阵 |
|---|---|
| 表面吸附能（覆盖度 ≤ 1/9） | ≥ 3×3 |
| 单空位缺陷 (bulk metal) | 4×4×4（≥ 64 atoms） |
| 单空位 (semiconductor / oxide) | ≥ 64 atoms 且最小镜像距离 > 8 Å |
| 二维缺陷 | ≥ 5×5 |
| 极化子 / 强局域缺陷 | 6×6×6 起步 |

**最小镜像距离**：缺陷之间至少 8 Å（金属）/ 10 Å（半导体）/ 12 Å（离子晶体）。

---

## 4. k 点估算 (KPOINTS, Γ-centered Monkhorst-Pack)

经验式：`k_i ≈ ceil(50 / a_i)`，单位 Å。

| 体系类型 | 加密因子 |
|---|---|
| 金属 | 1.0 (=50/a) |
| 半导体 | 0.6 (≈30/a) |
| 绝缘体 | 0.4 (≈20/a) |
| Slab 法线方向 | 强制 1 |
| 大盒子 (a > 20 Å) | 1×1×1 + Γ 即可 |

例：Pt(111) 3×3 supercell, a=b≈8.3 Å, c=40 Å → KPOINTS 6×6×1。

## 5. Packmol 密度（300 K, 1 bar）

| 物质 | 密度 (g/cm³) | 摩尔体积 (Å³/分子) |
|---|---|---|
| 水 H₂O | 1.000 | 29.9 |
| 重水 D₂O | 1.107 | 30.0 |
| 甲醇 CH₃OH | 0.792 | 67.2 |
| 乙醇 C₂H₅OH | 0.789 | 96.9 |
| 异丙醇 | 0.786 | 126.9 |
| 乙腈 CH₃CN | 0.786 | 86.7 |
| DMSO | 1.100 | 128.4 |
| DMF | 0.944 | 128.6 |
| THF | 0.889 | 134.5 |
| 苯 C₆H₆ | 0.876 | 148.0 |
| 甲苯 | 0.867 | 176.4 |
| 氯仿 CHCl₃ | 1.483 | 133.7 |
| 正己烷 | 0.655 | 217.2 |

**离子液体 / 熔盐**：必须查文献，不要外推。
**温度修正**：水 0 → 100°C 密度从 0.9998 → 0.9584；其他液体类似 ~0.1% / K，简单计算可忽略。

填充原则：`分子数 = 体积(Å³) × 密度(g/cm³) × NA / (摩尔质量 × 10²⁴)`。

---

## 6. Packmol 容差与盒子

- **容差 (tolerance)**：默认 2.0 Å（小分子）/ 2.5 Å（链状/环状大分子）/ 1.8 Å（极小分子且有 PBC 时）。
- **盒子收缩**：`box_size - 2 × tolerance` 是 Packmol 的有效填充体积，密度计算应基于此。
- **PBC**：Packmol 不显式支持 PBC，靠 tolerance 让分子离边界 ≥ 2 Å。后续在 LAMMPS/GROMACS 的 box 应等于 Packmol 盒子大小（不是减 tolerance 后的）。

---

## 7. 固液界面建模

**步骤模板**（Recipe pipeline）：
1. `bulk` → `slab` (按 §2 选层数) → `supercell` (按 §3 选大小)
2. `vacuum` 加大到 `slab + 水层 + 12 Å`
3. （表面修饰：B2 `saturate_dangling_bonds` 脚本，未实装则手动）
4. `filler` 在水层区域填水（§5 密度）
5. `selective_dynamics` 固定底层（§2）

**额外约束**：
- VASP：`LDIPOL=.TRUE., IDIPOL=3`（沿 c 轴的偶极校正）
- 含离子：先按浓度算离子数，再按电中性补反离子；密度算入水
- AIMD：水层 ≥ 10 Å 才能视作 bulk water

## 8. 缺陷 supercell

| 缺陷类型 | 起步 supercell | 镜像距离判据 |
|---|---|---|
| 单空位 (金属) | 4×4×4 (~64 atom) | > 8 Å |
| 单空位 (半导体) | ≥ 64 atom + 4×4×4 | > 10 Å |
| 替位掺杂 | 同空位 | 同上 |
| 间隙原子 | + 1 步收敛性 | > 10 Å |
| 双空位 / 团簇 | 5×5×5 起步 | > 12 Å |
| 表面缺陷 | in-plane 4×4 起步 | > 10 Å |

**电荷态**：带电缺陷必须做 finite-size 校正（FNV / Makov-Payne）；Recipe 只生成几何结构，校正参数在 vasp-incar / vasp-agent 处理。

---

## 9. 异质结 / 二维堆叠

- **失配率 (lattice mismatch)** ≤ 5%：可直接 supercell 配匹配；> 5% 需评估应变能或用 CSL 法
- **层间距**：vdW 层 (graphene/MoS₂ 上) 起步 3.3–3.5 Å；金属/氧化物界面 2.0–2.5 Å
- **极性匹配**：极性面贴极性面要看终止层电荷，避免极性灾难
- **真空层**：上下都加，每侧 ≥ 12 Å

## 10. 单位与命名约定

| 量 | 默认单位 |
|---|---|
| 长度 | Å |
| 能量 | eV |
| 温度 | K |
| 密度 | g/cm³ |
| 浓度 | mol/L（默认）/ wt% / mol%（必须显式） |
| 角度 | 度 (degree) |

**Recipe 内部所有数值默认上述单位**。如 LLM 收到 nm / kcal/mol / °C 输入，必须先换算再写入 Recipe，并在 metadata.notes 记录原始输入。

---

## 11. 何时不能用默认

下面这些场景必须主动询问用户，**不要套默认**：

- 高压 / 低温体系（密度、晶格常数偏离明显）
- 磁性体系（`ISPIN` / 初始磁矩在 vasp-incar，但 Recipe 选 supercell 时要避开反铁磁周期）
- 含重元素（>Z=82）的相对论效应
- 强关联体系（DFT+U / hybrid）
- 范德瓦尔斯主导（层间距、堆叠方式）
- 用户明确说"我要复现某文献" → 用文献参数
