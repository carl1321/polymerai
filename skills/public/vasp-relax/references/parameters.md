# vasp-relax — 参数指南

## 核心参数

### ISIF — 哪些自由度在动
| 值 | 计算量 | 位置 | 形状 | 体积 | 场景 |
|---|---|---|---|---|---|
| 1 | 力 | ✓ | ✗ | ✗ | 固定晶胞弛豫原子（slab、defect supercell） |
| 2 | 力+应力 | ✓ | ✗ | ✗ | 同上，但报告应力 |
| 3 | 力+应力 | ✓ | ✓ | ✓ | 全自由弛豫（默认、体相） |
| 4 | 力+应力 | ✓ | ✓ | ✗ | 等体积变形状（磁性相变研究） |
| 5 | 力+应力 | ✗ | ✓ | ✗ | 固定原子，只变形状 |
| 7 | 力+应力 | ✗ | ✗ | ✓ | 只变体积（体弹模量） |

默认 `ISIF=3`。Slab / defect supercell 要改成 `ISIF=2`，否则真空层会被压缩。

### IBRION — 离子弛豫算法
| 值 | 算法 | 典型用途 |
|---|---|---|
| -1 | 不动离子 | SCF only |
| 1 | RMM-DIIS | 接近极小时高效 |
| 2 | CG | 远离极小时稳健（默认） |
| 3 | 阻尼 MD | 极难收敛的体系、过渡态附近 |
| 5/6 | 有限位移 phonon | vasp-phonon 用，不用这里 |
| 8 | DFPT phonon | 同上 |

### EDIFFG — 离子收敛判据
- **负值 (eV/Å)**：力 < |EDIFFG|，通常用 `-0.02`（松）/`-0.01`（紧）
- **正值 (eV)**：两步能量差 < EDIFFG（不推荐单独用）

### NSW — 最大离子步数
默认 100。若 `IBRION=3` 或超胞很大，可以调到 200+。

### ENCUT
不设则从 POTCAR 的 ENMAX 最大值 × 1.3 自动取。收敛测试建议手动指定。

## 常见陷阱

1. **Slab / 2D 材料**：ISIF=3 会压缩真空层 → 改 ISIF=2
2. **磁性体系第一次弛豫**：先做 ISIF=2 固定晶胞、再 ISIF=3
3. **收敛到鞍点（虚频）**：加微扰后重跑，或用 IBRION=1 切换
4. **ZBRENT fatal error**：已由 handler 自动改 IBRION=1, POTIM=0.1
5. **体积变化过大**：ENCUT 偏低导致 Pulay 应力；ENCUT 提到 1.3×ENMAX 以上

## 何时用 vasp-incar 生成而非依赖默认

- 有磁性初值要求（MAGMOM）
- HSE / meta-GGA
- 高精度（EDIFFG ≤ -0.005）
- 包含 vdW（IVDW）
